How to get your credentials.json for Gmail API access

To obtain your credentials.json file, follow these steps:

  1. Go to the Google Cloud Console API Credentials page.
  2. Select an existing project or create a new one.
  3. Click + CREATE CREDENTIALS and choose OAuth client ID.
  4. For "Application type", select Desktop app (or Web application if deploying on a server).
  5. Click CREATE.
  6. A dialog will appear with your client ID and client secret. Click DOWNLOAD JSON.
  7. Rename the downloaded file to credentials.json and upload it here.
  8. Ensure the Gmail API is enabled for your project: Go to Gmail API in API Library and click "Enable".

Important: When you run the application for the first time with new credentials, a browser window will open asking you to authorize access to your Gmail account. You will need to grant permission.

How to get your OpenAI API Key

To obtain your OpenAI API Key, follow these steps:

  1. Go to the OpenAI API Keys page.
  2. If you don't have an account, you'll need to create one.
  3. Click on the + Create new secret key button.
  4. Give your key a name (optional) and click Create secret key.
  5. Copy the key that appears. It will look something like sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx.
  6. Paste this key into the "OpenAI API Key" field on this page.

Security Warning: Treat your API key like a password. Do not share it publicly or commit it to version control.
